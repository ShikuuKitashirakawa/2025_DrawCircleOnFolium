# 2025_DrawCircleOnFolium

Streamlit と GitHub の連携および Folium を用いた地図描画の学習用リポジトリです。

## 概要 (Description)
このプロジェクトは、以下の技術スタックを習得するためのテストコードを管理しています。
- **Streamlit**: Webアプリケーションの構築とデプロイ
- **GitHub**: バージョン管理とリモートリポジトリの操作
- **Folium**: 地図上への図形（円）の描画と表示
